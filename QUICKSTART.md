# 快速启动指南

## 使用局域网聊天功能

### 第一步：安装 Go 环境
如果还没有安装 Go，请访问 https://golang.org/dl/ 下载安装。

### 第二步：下载依赖
```bash
cd DataToolbox
go mod download
```

### 第三步：构建服务器
```bash
# Windows
go build -o DataToolbox-Server.exe server.go

# Linux/Mac
go build -o DataToolbox-Server server.go
```

### 第四步：启动服务器
```bash
# Windows
DataToolbox-Server.exe

# Linux/Mac
./DataToolbox-Server
```

启动成功后会显示：
```
============================================================
DataToolbox 服务器已启动
============================================================
本地访问: http://localhost:8080
局域网访问: http://192.168.1.xxx:8080
外网访问: http://<your-public-ip>:8080
============================================================
功能: 文件服务器 + 局域网聊天
============================================================
按 Ctrl+C 停止服务器
============================================================
```

### 第五步：访问聊天应用

1. 在本机浏览器打开: `http://localhost:8080`
2. 点击"联机"分区
3. 点击"局域网聊天"应用
4. 设置你的昵称

### 第六步：连接其他设备

1. 确保其他设备在同一局域网
2. 在其他设备浏览器打开: `http://<服务器IP>:8080/apps/lan-chat/`
3. 设置昵称后，双方即可相互发现
4. 选择联系人，开始聊天！

## 功能演示

### 发送文字
直接在输入框输入，按 Enter 发送

### 发送表情
点击😊按钮，选择表情

### 发送图片
点击🖼️按钮，选择图片文件

### 发送文件
点击📎按钮，选择任意文件

## 注意事项

1. 必须启动服务器版本，直接打开HTML文件无效
2. 所有设备必须在同一局域网内
3. 确保防火墙允许8080端口（或自定义端口）
4. 消息仅保存在浏览器内存，刷新后丢失

## 故障排查

### 问题：无法启动服务器
- 检查Go环境是否安装正确：`go version`
- 检查端口8080是否被占用
- 尝试使用其他端口：`DataToolbox-Server.exe -port 9000`

### 问题：其他设备无法连接
- 检查防火墙设置
- 确认局域网IP地址是否正确
- 尝试ping服务器IP

### 问题：提示需要服务器版本
- 必须通过服务器访问，不能直接打开HTML文件
- 确保访问的URL是 `http://IP:端口/apps/lan-chat/`
