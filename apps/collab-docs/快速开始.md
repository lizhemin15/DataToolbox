# 快速开始 - 在线协作文档

## 🚀 启动步骤

### 1️⃣ 编译服务器（首次使用）
```bash
# 在项目根目录执行
go build -o DataToolbox-Server.exe server.go
```

### 2️⃣ 启动服务器
```bash
# Windows
.\DataToolbox-Server.exe

# Mac/Linux
./DataToolbox-Server
```

### 3️⃣ 访问应用
打开浏览器访问：
- **本地访问**: http://localhost:8080/apps/collab-docs/
- **局域网访问**: http://你的局域网IP:8080/apps/collab-docs/

## 📝 快速体验

### 创建第一个协作文档
1. 点击"新建文档"按钮
2. 输入文档名称（如"测试表格"）
3. 选择文档类型（Excel或Word）
4. 点击"创建"按钮

### 多人协作
1. 在同一局域网的其他设备上打开相同网址
2. 点击同一个文档卡片
3. 开始实时协作编辑
4. 可以看到其他用户的光标位置

## 💡 使用技巧

### Excel协作
- 点击单元格直接编辑
- 黄色边框表示其他用户正在编辑的单元格
- 使用"添加行/列"按钮扩展表格
- 点击"导出Excel"下载完整表格

### Word协作
- 使用顶部工具栏设置文字格式
- 实时看到其他用户的编辑
- 支持插入图片和链接
- 点击"导出Word"保存文档

### 协作提示
- 左上角点击用户名可修改昵称
- 右上角显示当前在线编辑人数
- 避免同时编辑同一位置，防止内容冲突

## ⚠️ 重要提示

1. **必须通过服务器运行**：不能直接双击打开HTML文件
2. **数据保存**：当前版本数据存储在服务器内存，重启后会丢失
3. **网络要求**：所有用户必须在同一局域网内
4. **定期导出**：重要内容请及时导出备份

## 🐛 遇到问题？

### 提示"需要服务器版本"
- 确认服务器已启动
- 通过 http://localhost:8080 访问，不要直接打开文件

### 看不到其他用户
- 确认所有人在同一局域网
- 检查防火墙是否阻止端口8080
- 尝试刷新页面

### 内容不同步
- 检查网络连接
- 查看浏览器控制台是否有错误
- 尝试重新打开文档
