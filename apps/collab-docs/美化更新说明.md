## 🚀 性能优化

### CSS性能
- 使用 `transform` 和 `opacity` 做动画
- GPU加速（`will-change`）
- 避免重排（`transform` 代替 `top/left`）

### 动画优化
- 使用 `cubic-bezier` 缓动函数
- 动画时长控制在 0.3-0.5s
- 避免同时运行过多动画
