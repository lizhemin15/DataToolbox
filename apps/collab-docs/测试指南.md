# 美化功能测试指南

## 🧪 测试环境准备

### 1. 重新编译服务器
```bash
cd z:\Documents\GitHub\DataToolbox
go build -o DataToolbox-Server.exe server.go
```

### 2. 启动服务器
```bash
.\DataToolbox-Server.exe
```

### 3. 打开浏览器
- 浏览器A: `http://localhost:8080/apps/collab-docs/`
- 浏览器B: `http://localhost:8080/apps/collab-docs/`（隐身模式或不同浏览器）

## 🐛 常见问题排查

### 问题1：用户名不更新
**检查**：
- WebSocket是否连接
- 浏览器控制台是否有错误
- 服务器是否正常运行

### 问题2：光标不显示
**检查**：
- 两个用户是否在同一文档
- 是否点击了不同的单元格
- 刷新页面重试

### 问题3：头像不显示
**检查**：
- 用户名是否已设置
- CSS是否正确加载
- 浏览器控制台检查错误

### 问题4：动画卡顿
**检查**：
- 电脑性能
- 浏览器版本（建议Chrome/Edge 90+）
- 关闭其他标签页

### 问题5：在线人数不准确
**检查**：
- WebSocket连接状态
- 服务器日志
- 刷新页面
