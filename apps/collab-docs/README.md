# 在线协作文档

## 功能特性

### 文档管理
- ✅ 文档列表展示
- ✅ 新建Excel/Word文档
- ✅ 删除文档
- ✅ 文档筛选（全部/Excel/Word）

### 实时协作
- ✅ 多人同时编辑
- ✅ 实时内容同步
- ✅ 在线用户显示
- ✅ 光标位置显示

### Excel编辑器
- ✅ 表格编辑
- ✅ 添加行/列
- ✅ 实时单元格同步
- ✅ 导出Excel文件

### Word编辑器
- ✅ 富文本编辑
- ✅ 文字格式化（标题、加粗、斜体等）
- ✅ 颜色和背景色
- ✅ 列表和对齐
- ✅ 插入链接和图片
- ✅ 导出Word文件

## 重要说明

**⚠️ 本应用必须通过服务器版本运行，不支持直接打开HTML文件！**

在线协作文档需要WebSocket服务器支持实时通信功能，直接打开HTML文件将无法正常使用。

## 使用方法

### 1. 启动服务器
```bash
# 进入项目目录
cd DataToolbox

# 启动服务器（需要先编译server.go）
./DataToolbox-Server
```

### 2. 访问应用
在浏览器中打开：
- 本地: `http://localhost:8080/apps/collab-docs/`
- 局域网: `http://<局域网IP>:8080/apps/collab-docs/`

### 3. 新建文档
1. 点击"新建文档"按钮
2. 输入文档名称
3. 选择文档类型（Excel或Word）
4. 点击"创建"

### 4. 编辑文档
#### Excel文档
- 点击单元格即可编辑
- 使用工具栏添加行或列
- 其他用户正在编辑的单元格会高亮显示
- 点击"导出Excel"可下载xlsx文件

#### Word文档
- 使用顶部工具栏进行文字格式化
- 支持标题、列表、对齐等丰富格式
- 实时看到其他用户的编辑
- 点击"导出Word"可下载doc文件

### 5. 协作编辑
- 多个用户可同时打开同一文档
- 右上角显示当前在线编辑人数
- 可以看到其他用户的光标位置

### 6. 管理文档
- 点击文档卡片打开文档
- 在编辑页面点击"删除"按钮删除文档
- 点击文档标题可修改文档名称

## 技术实现

### 前端
- 原生 JavaScript
- WebSocket 实时通信
- SheetJS (xlsx) - Excel处理
- Quill.js - 富文本编辑器

### 后端
- Go + gorilla/websocket
- 内存存储文档数据
- 实时消息广播

### 通信协议
```javascript
// 请求文档列表
{ type: 'doc-list-request' }

// 返回文档列表
{ type: 'doc-list', documents: [...] }

// 创建文档
{ 
  type: 'doc-created',
  document: {
    id: '文档ID',
    title: '文档名称',
    type: 'excel|word',
    creator: '创建者',
    createdAt: 时间戳,
    content: {}
  }
}

// 删除文档
{ type: 'doc-deleted', docId: '文档ID' }

// 打开文档
{ type: 'doc-opened', docId: '文档ID', document: {...} }

// 更新文档内容
{
  type: 'doc-update',
  docId: '文档ID',
  update: {
    type: 'excel-cell|excel-add-row|word-delta',
    ...更新数据
  }
}

// 更新文档标题
{ type: 'doc-title-update', docId: '文档ID', title: '新标题' }

// 光标位置
{
  type: 'doc-cursor',
  docId: '文档ID',
  cursor: { type: 'excel|word', row: 0, col: 0 }
}
```

## 使用技巧

### Excel编辑
1. **快速导航**：Tab键移动到下一个单元格
2. **批量添加**：连续点击"添加行/列"按钮
3. **实时协作**：黄色边框标识其他用户正在编辑的单元格

### Word编辑
1. **快捷键**：
   - Ctrl+B：加粗
   - Ctrl+I：斜体
   - Ctrl+U：下划线
2. **格式刷**：选中文字后使用工具栏快速设置格式
3. **插入图片**：点击图片按钮上传本地图片

### 协作技巧
1. **设置昵称**：点击左上角用户名修改，便于团队识别
2. **避免冲突**：观察其他用户光标位置，避免同时编辑同一位置
3. **定期导出**：编辑完成后及时导出备份

## 注意事项

1. **数据存储**
   - 文档数据存储在服务器内存中
   - 服务器重启后数据会丢失
   - 建议定期导出重要文档

2. **网络要求**
   - 必须在同一局域网内
   - 稳定的网络连接确保实时同步

3. **并发编辑**
   - 支持多人同时编辑不同位置
   - 同时编辑同一位置可能导致内容覆盖
   - 建议团队协调编辑区域

4. **文件大小**
   - Excel：建议不超过1000行
   - Word：建议内容不超过10000字
   - 大文档可能影响同步性能

5. **浏览器兼容**
   - 推荐使用现代浏览器
   - Chrome、Edge、Firefox、Safari
   - 需要支持WebSocket和ES6

## 常见问题

### Q: 为什么提示需要服务器版本？
A: 在线协作文档必须通过服务器版本运行，不支持直接打开HTML文件。请：
1. 编译并启动服务器：`go build -o DataToolbox-Server.exe server.go`
2. 通过浏览器访问：`http://localhost:8080/apps/collab-docs/`

### Q: 为什么看不到其他用户？
A: 确保：
- 服务器已启动
- 所有用户在同一局域网
- 所有用户已打开应用页面

### Q: 文档修改不同步？
A: 检查：
- WebSocket连接是否正常
- 浏览器控制台是否有错误
- 网络连接是否稳定

### Q: 导出的Excel格式不对？
A: Excel导出使用xlsx格式，可能需要用Excel或WPS打开，某些在线表格工具可能不兼容。

### Q: Word导出后格式丢失？
A: 导出的doc文件是HTML格式，建议用Word打开后另存为标准docx格式。

### Q: 服务器重启后文档丢失？
A: 当前版本数据存储在内存中，服务器重启会清空。建议在后续版本中添加持久化存储。
