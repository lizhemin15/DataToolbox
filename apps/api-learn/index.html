关于本课程

本课程通过闯关式的方式，教你如何在信息池系统中编写 API 接口。信息池是一个 API 管理系统，通过 MyBatis 语法定义 SQL 查询，配合接口参数传参，在数据库中执行查询或其他操作后返回 JSON 结果。你将学会从最简单的查询接口到复杂的动态 SQL 接口的完整编写方法。

⚠️

前置课程

请先完成以下课程：SQL语法学习、MyBatis语法学习、网络请求教学，掌握 SQL 查询、MyBatis 参数映射和 HTTP 请求基础后再学习本课程，效果更佳。
