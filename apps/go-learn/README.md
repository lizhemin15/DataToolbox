# Go语言学习

闯关式学习Go语言编程，从Hello World到并发编程，循序渐进掌握Go核心特性。

## ⚠️ 重要提示

**此应用只能在服务端使用**，需要满足以下条件：
1. 服务器端必须安装 Go 环境（Go 1.16+）
2. 必须通过本项目的服务器运行（`server.exe`）
3. 代码在服务器端编译执行，请勿输入恶意代码
4. 执行超时限制为 10 秒

## 功能特点

- 🎮 **闯关式学习**：30个精心设计的关卡，循序渐进
- 🔓 **关卡解锁**：完成前一关才能解锁下一关，确保学习质量
- ✏️ **实时编辑**：内置Go代码编辑器
- ▶️ **即时运行**：在浏览器中运行Go代码，实时查看结果
- ✅ **自动验证**：提交代码后自动验证输出是否正确
- 💡 **提示系统**：每个关卡都有提示帮助你完成挑战
- 📊 **进度跟踪**：记录你的学习进度

## 学习路线

### 基础入门（关卡 1-5）
1. **Hello World** - 你的第一个Go程序
2. **变量声明** - 学习如何声明和使用变量
3. **基本运算** - 学习算术运算
4. **条件语句 if** - 学习使用if条件判断
5. **for 循环** - 学习使用for循环

### 函数与数据结构（关卡 6-9）
6. **函数定义** - 学习如何定义和调用函数
7. **数组** - 学习数组的使用
8. **切片 Slice** - 学习切片的使用
9. **Map 映射** - 学习map的使用

### 面向对象（关卡 10-12）
10. **结构体 Struct** - 学习结构体的定义和使用
11. **方法** - 学习为结构体定义方法
12. **接口 Interface** - 学习接口的定义和实现

### 高级特性（关卡 13-15）
13. **错误处理** - 学习Go的错误处理机制
14. **Goroutine 并发** - 学习Go的并发编程
15. **Channel 通道** - 学习使用channel进行通信

### 进阶特性（关卡 16-20）
16. **指针 Pointer** - 学习指针的使用
17. **defer 延迟执行** - 学习defer语句
18. **panic 和 recover** - 学习异常处理
19. **字符串处理** - 学习strings包的使用
20. **包和导入** - 学习自定义包的使用

### 文件与数据（关卡 21-24）
21. **文件读取** - 学习读取文件
22. **文件写入** - 学习写入文件
23. **JSON 编码** - 学习JSON序列化
24. **JSON 解码** - 学习JSON反序列化

### Web与网络（关卡 25-26）
25. **HTTP 客户端** - 学习发送HTTP请求
26. **HTTP 服务器** - 学习创建HTTP服务器

### 实战技能（关卡 27-30）
27. **命令行参数** - 学习处理命令行参数
28. **环境变量** - 学习读取环境变量
29. **时间处理** - 学习time包的使用
30. **编译和打包** - 学习Go程序的编译部署

## 技术实现

### 前端
- 纯HTML + CSS + JavaScript实现
- 响应式设计，支持移动设备
- 本地存储学习进度

### 后端
- Go语言编写的HTTP服务器
- `/api/run-go` API端点执行用户提交的代码
- 临时文件管理和安全控制
- 10秒超时保护

## 使用方法

1. 启动服务器：`./server.exe`
2. 在浏览器中打开应用
3. 从第一关开始学习（其他关卡处于锁定状态）
4. 阅读教程和任务，参考示例代码
5. 编写代码并运行测试
6. 提交答案验证结果
7. 通关后自动解锁下一关
8. 依次完成所有30个关卡

### 关卡状态说明

- 🔒 **锁定** - 灰色显示，需要完成前一关才能解锁
- 💙 **可玩** - 蓝色光晕，当前可以挑战的关卡
- ✅ **已完成** - 绿色边框，显示完成标记

## 安全说明

- 建议仅在受信任的内网环境中使用

- 学习进度保存在浏览器本地存储中
