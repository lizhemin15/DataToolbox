# 数据本体池

数据库管理、数据治理、本体论抽象、接口分发、AI助手的一体化数据平台。

## 功能模块

### 已实现

1. **用户鉴权**
   - 登录系统
   - Token认证
   - 默认管理员账号：admin / admin1234

2. **数据库管理**
   - MySQL数据库接入
   - 连接测试
   - 保存数据库配置
   - 查看数据库表列表
   - 预览表数据（最多100条）

### 待实现

3. **数据治理**
4. **本体论抽象**
5. **接口分发**
6. **AI助手**

## 使用说明

### 1. 登录系统

首次使用请使用默认管理员账号登录：
- 用户名：`admin`
- 密码：`admin1234`

### 2. 添加MySQL数据库

1. 点击左侧"+ 添加"按钮
2. 填写数据库配置信息：
   - 数据库名称：自定义名称（用于识别）
   - 主机地址：如 localhost 或 IP地址
   - 端口：默认 3306
   - 用户名：MySQL用户名
   - 密码：MySQL密码
   - 数据库名：要连接的数据库名
3. 点击"测试连接"验证配置
4. 点击"保存"保存配置

### 3. 查看数据库

1. 在左侧列表中点击已添加的数据库
2. 右侧将显示：
   - 数据库基本信息
   - 连接状态
   - 数据表列表

### 4. 预览表数据

1. 在数据表列表中点击要查看的表
2. 下方将显示表的前100条数据

## 技术架构

### 前端
- HTML5 + CSS3 + JavaScript
- 纯原生实现，无额外依赖

### 后端
- Go 语言
- MySQL驱动：github.com/go-sql-driver/mysql
- UUID生成：github.com/google/uuid

## 安全说明

- 密码使用MD5哈希存储
- API使用Token进行身份验证
- 数据库密码加密存储在服务器内存中
- 建议在生产环境中修改默认管理员密码
