欢迎使用AI助手

                            
使用 @数据库名 来引用数据库，AI将根据您的问题生成SQL查询并返回结果

                            

                                
💡 智能上下文：

                                

                                    • 首次提问需要使用 @数据库名

                                    • 后续对话会记住数据库，无需重复 @

                                    • 使用新的 @ 会切换到新数据库

                                    • 点击右上角 ✕ 可清除上下文
                                

                                
📝 查询示例：

                                

                                    • "@数据库 有哪些表？"

                                    • "users表有哪些字段？" ← 继续使用之前的数据库

                                    • "查看products表的前10条数据"

                                    • "查询价格大于100的产品"
                                

                                
🔌 创建接口示例：

                                

                                    • "@数据库 帮我创建一个查询用户的接口"

                                    • "生成一个获取产品列表的API，按价格排序"

                                    • "创建接口：根据ID查询订单详情"
