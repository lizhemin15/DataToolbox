# 局域网聊天 - 5分钟快速上手

## 第一步：启动服务器（30秒）

### Windows
```bash
DataToolbox-Server.exe
```

### Linux/Mac
```bash
./DataToolbox-Server
```

看到以下提示说明启动成功：
```
============================================================
DataToolbox 服务器已启动
============================================================
本地访问: http://localhost:8080
局域网访问: http://192.168.1.xxx:8080
============================================================
```

## 第二步：打开聊天应用（10秒）

### 本机测试
打开浏览器访问：`http://localhost:8080/apps/lan-chat/`

### 局域网设备
其他设备访问：`http://<服务器IP>:8080/apps/lan-chat/`

> 💡 提示：服务器IP在启动信息中显示

## 第三步：设置昵称（5秒）

1. 点击左上角的用户名
2. 输入你的昵称
3. 确定

## 第四步：发送第一条消息（10秒）

### 打开第二个浏览器标签页
访问同样的地址，模拟第二个用户

### 发送消息
1. 在左侧列表看到对方
2. 点击对方头像
3. 输入"你好"
4. 按 Enter 发送

✅ 如果对方收到了消息，说明基本功能正常！

## 第五步：尝试发送文件（1分钟）

### 方法1：粘贴图片（最快）
1. 截图（Win+Shift+S 或其他截图工具）
2. 在输入框按 `Ctrl+V`
3. 预览图片
4. 点击"发送"

### 方法2：拖拽文件（最方便）
1. 打开文件管理器
2. 选择一个文件或图片
3. 拖拽到聊天窗口
4. 确认后发送

### 方法3：点击按钮（传统方式）
1. 点击 🖼️ 按钮发送图片
2. 点击 📎 按钮发送文件
3. 点击 😊 按钮发送表情

## 多设备连接

### 手机连接
1. 确保手机和电脑在同一WiFi
2. 打开手机浏览器
3. 访问：`http://<电脑IP>:8080/apps/lan-chat/`
4. 开始聊天

### 多台电脑连接
1. 在另一台电脑打开浏览器
2. 访问：`http://<服务器IP>:8080/apps/lan-chat/`
3. 设置昵称
4. 开始聊天

## 故障排查

### 问题1：看不到其他用户
**检查：**
- ✅ 服务器是否启动？
- ✅ 在同一局域网？
- ✅ 两个设备都打开了聊天页面？

**解决：**
按 F12 打开控制台，查看是否有错误信息

### 问题2：无法发送消息
**检查：**
- ✅ 对方是否在线？
- ✅ 状态指示灯是否为绿色？
- ✅ 网络连接是否正常？

**解决：**
刷新页面重新连接

### 问题3：粘贴图片无效
**检查：**
- ✅ 剪贴板是否有图片？
- ✅ 浏览器是否为现代浏览器？
- ✅ 是否在输入框内粘贴？

**解决：**
使用拖拽方式发送，或点击按钮发送

### 问题4：文件传输失败
**检查：**
- ✅ 文件是否超过大小限制？
- ✅ 网络是否稳定？

**解决：**
- 图片限制：10MB
- 文件限制：50MB
- 大文件建议使用其他方式传输

## 性能优化建议

### 传输大文件
1. 确保局域网连接稳定
2. 避免同时发送多个大文件
3. 大文件（>50MB）建议用U盘或其他方式

### 长时间使用
1. 定期刷新页面释放内存
2. 关闭不必要的聊天窗口
3. 清理浏览器缓存

### 多人使用
1. 建议同时在线用户不超过20人
2. 避免频繁发送大量消息
3. 合理使用文件传输功能
