# 局域网聊天

### 即时通讯
- ✅ 实时消息传输（WebSocket）
- ✅ 自动设备发现
- ✅ 在线状态显示
- ✅ 已读未读状态

### 消息类型
- ✅ 文字消息
- ✅ 表情符号
- ✅ 图片发送（支持预览）
- ✅ 文件传输（任意格式）

### 界面设计
- ✅ 仿微信UI
- ✅ 响应式布局
- ✅ 流畅动画效果
- ✅ 消息气泡样式

### 用户体验
- ✅ 自定义昵称
- ✅ 消息搜索
- ✅ 消息时间显示
- ✅ 系统消息提醒

## 使用方法

### 1. 启动服务器
```bash
# 进入项目目录
cd DataToolbox

# 启动服务器（需要先编译server.go）
./DataToolbox-Server
```

### 2. 访问聊天应用
在浏览器中打开：
- 本地: `http://localhost:8080/apps/lan-chat/`
- 局域网: `http://<局域网IP>:8080/apps/lan-chat/`

### 3. 设置昵称
- 点击左上角的用户名可以修改昵称
- 昵称会保存在本地，下次自动加载

### 4. 开始聊天
- 等待其他设备连接到同一局域网
- 在左侧聊天列表中选择联系人
- 开始发送消息、图片或文件

## 功能说明

### 发送消息
- 在输入框中输入文字，按 Enter 发送
- Shift + Enter 换行

### 发送表情
- 点击工具栏的表情按钮😊
- 在弹出的表情选择器中选择表情

### 发送图片
**方式1：点击按钮**
- 点击工具栏的图片按钮🖼️
- 选择图片文件

**方式2：粘贴图片**（推荐）
- 在聊天界面复制图片（Ctrl+C）
- 在输入框粘贴（Ctrl+V）
- 预览后点击"发送"

**方式3：拖拽图片**（推荐）
- 从文件管理器拖拽图片到聊天窗口
- 预览后点击"发送"

### 发送文件
**方式1：点击按钮**
- 点击工具栏的文件按钮📎
- 选择任意文件

**方式2：拖拽文件**（推荐）
- 从文件管理器拖拽文件到聊天窗口
- 确认文件信息后点击"发送"
- 接收方点击文件卡片即可下载

- 原生 JavaScript

### 后端

## 使用技巧

2. **拖拽发送**：从文件管理器直接拖拽文件到聊天窗口
3. **批量发送**：选中多个文件一起拖拽，可以批量发送

- `Enter` - 发送消息

- `Ctrl + V` - 粘贴图片

## 注意事项

1. **网络要求**
   - 必须在同一局域网内
   - 确保防火墙允许端口访问

2. **文件大小限制**
   - 图片：单个不超过 10MB
   - 文件：单个不超过 50MB
   - 大文件传输可能较慢，请耐心等待

3. **隐私安全**
   - 消息不会保存到服务器
   - 所有数据仅存储在本地浏览器
   - 关闭浏览器后消息记录丢失
   - 局域网传输相对安全

4. **浏览器兼容**
   - 推荐使用现代浏览器
   - Chrome、Edge、Firefox、Safari
   - 粘贴功能需要浏览器支持 Clipboard API

## 常见问题

### Q: 为什么看不到其他用户？
A: 确保：
- 服务器已启动
- 其他设备在同一局域网
- 其他设备已打开聊天页面

### Q: 为什么提示需要服务器版本？
A: 局域网聊天必须通过服务器版本运行，不支持直接打开HTML文件。

### Q: 消息能否保存？
A: 当前版本消息仅保存在浏览器内存中，刷新或关闭页面后会丢失。

### Q: 支持群聊吗？
A: 当前版本仅支持点对点单聊，未来可扩展群聊功能。
