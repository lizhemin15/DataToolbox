# 局域网聊天 - 功能特性详解

## 🎯 核心功能

### 1. 实时消息传输
- ✅ WebSocket 长连接，毫秒级延迟
- ✅ 自动设备发现，无需配置
- ✅ 在线状态实时显示
- ✅ 消息已读未读状态

### 2. 多种消息类型
- ✅ 文字消息
- ✅ 表情符号（80+个常用表情）
- ✅ 图片消息（支持预览和下载）
- ✅ 文件传输（支持任意格式）

## ✨ 特色功能

### 📋 粘贴发送（推荐）

**使用场景：**
- 截图后快速分享
- 从网页复制图片后发送
- 从剪贴板发送图片

**操作步骤：**
1. 使用任意截图工具截图（如 Win+Shift+S）
2. 或从网页/文件管理器复制图片（Ctrl+C）
3. 在聊天输入框按 `Ctrl+V` 粘贴
4. 预览图片
5. 点击"发送"按钮

**优势：**
- ⚡ 最快速的图片分享方式
- 🎯 无需保存临时文件
- 💡 工作流程最自然

### 🖱️ 拖拽发送（推荐）

**单文件拖拽：**
1. 从文件管理器选中文件
2. 拖拽到聊天窗口
3. 预览文件信息
4. 点击"发送"按钮

**多文件拖拽：**
1. 从文件管理器选中多个文件（Ctrl+点击或框选）
2. 拖拽到聊天窗口
3. 预览所有文件列表
4. 点击"全部发送"

**支持的拖拽效果：**
- 📁 拖拽到聊天区域时，背景变绿色
- 💬 显示"松开鼠标发送文件"提示
- 🎨 流畅的动画效果

### 🔘 按钮发送（传统方式）

**发送图片：**
- 点击 🖼️ 按钮 → 选择图片 → 自动发送

**发送文件：**
- 点击 📎 按钮 → 选择文件 → 自动发送

**发送表情：**
- 点击 😊 按钮 → 选择表情 → 插入到输入框

3. 查看未读消息红点提示

## 📝 注意事项

### 使用限制
1. **必须启动服务器**：不能直接打开HTML文件
2. **同一局域网**：所有设备必须在同一网络
3. **浏览器要求**：现代浏览器（Chrome/Edge/Firefox/Safari）
4. **文件大小**：注意文件大小限制，避免传输超时

### 常见问题
1. **看不到其他用户**：检查是否在同一局域网，服务器是否启动
2. **消息发送失败**：检查对方是否在线，网络是否正常
3. **文件传输失败**：检查文件大小是否超限，网络是否稳定
4. **粘贴不生效**：检查剪贴板是否有图片，浏览器权限是否允许
