# 🎯 拖拽功能说明

## 功能概述

文档编辑器现在支持**思维导图节点**和**图表图形**的拖拽功能！

---

## 🧠 思维导图拖拽

### 功能特性

#### 1. **拖拽节点**
- 鼠标拖住节点即可移动位置
- 拖拽时连线自动跟随更新
- 支持任意节点的自由移动

#### 2. **缩放和平移**
- **鼠标滚轮**: 放大/缩小视图
- **拖拽空白区域**: 平移整个思维导图
- **缩放范围**: 0.5x - 3x

#### 3. **节点交互**
- **单击**: 选中节点（变为橙色高亮）
- **双击**: 编辑节点内容
- **拖拽**: 移动节点位置

#### 4. **视觉反馈**
- 选中节点显示橙色边框
- 拖拽时光标变为移动图标
- 连线实时跟随节点移动

### 操作方法

```
拖拽节点：      按住鼠标左键拖动节点
编辑内容：      双击节点
选中节点：      单击节点
缩放视图：      鼠标滚轮
平移视图：      拖拽空白区域
添加子节点：    选中父节点 → 点击"➕ 添加节点"
删除节点：      选中节点 → 点击"🗑️ 删除节点"
重置视图：      点击"🔄 重置视图"
```

## 📐 图表编辑器拖拽

### 功能特性

#### 1. **拖拽图形**
- 点击图形即可选中并拖拽
- 拖拽到任意位置
- 支持所有图形类型

#### 2. **图形选择**
- **单击图形**: 选中（变为绿色）
- **点击空白处**: 取消选中
- **选中状态**: 显示虚线选择框

#### 3. **图形删除**
- **双击图形**: 删除确认对话框
- 删除后自动保存到历史记录

#### 4. **智能光标**
- 悬停在图形上：光标变为移动图标
- 拖拽时：光标保持移动状态
- 空白区域：默认光标

### 支持的图形

所有图形都支持拖拽：
- 📦 矩形
- ⭕ 椭圆
- ◆ 菱形
- 🔺 三角形
- ➡️ 箭头

### 操作方法

```
选中图形：      单击图形
拖拽图形：      按住鼠标左键拖动
删除图形：      双击图形 → 确认删除
取消选中：      点击空白区域
撤销操作：      点击"↶ 撤销"
清空画布：      点击"🗑️ 清空"
```

### 视觉效果

| 状态 | 颜色 | 边框 | 说明 |
|------|------|------|------|
| 未选中 | 蓝色 | 深蓝边框 (2px) | 默认状态 |
| 选中 | 绿色 | 深绿边框 (3px) | 可拖拽状态 |
| 选择框 | - | 橙色虚线框 | 突出显示 |

## 🎨 技术实现

### 思维导图 (D3.js)

```javascript
// 拖拽功能
const dragHandler = d3.drag()
    .on('start', ...)  // 开始拖拽
    .on('drag', ...)   // 拖拽中
    .on('end', ...);   // 结束拖拽

// 缩放功能
const zoom = d3.zoom()
    .scaleExtent([0.5, 3])
    .on('zoom', ...);
```

### 图表编辑器 (Canvas)

```javascript
// 鼠标事件
canvas.onmousedown  // 选中图形
canvas.onmousemove  // 拖拽移动
canvas.onmouseup    // 结束拖拽
canvas.ondblclick   // 删除图形

// 碰撞检测
isPointInShape(x, y, shape)
```

## 🔧 历史记录

### 思维导图
- 节点移动会保存位置信息
- 可通过"重置视图"恢复初始布局

### 图表编辑器
- 每次拖拽结束自动保存到历史
- 支持撤销操作（点击"↶ 撤销"）
- 历史记录保存所有形状状态

## 💡 使用技巧

### 思维导图
1. **快速布局**: 先用默认布局生成，再拖拽调整
2. **层次清晰**: 拖拽时保持连线整洁
3. **放大编辑**: 缩放到合适大小进行精细调整
4. **批量移动**: 平移整个视图调整显示区域

### 图表编辑器
1. **精确定位**: 拖拽到精确位置再松开
2. **快速删除**: 双击不需要的图形
3. **撤销保底**: 出错了用撤销功能
4. **分组摆放**: 将相关图形摆放在一起
