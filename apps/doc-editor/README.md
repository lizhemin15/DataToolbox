# 文档编辑器

在线编辑各类办公文档的综合工具。

## 功能特性

### 📊 表格编辑
- **x-spreadsheet**: 轻量级表格编辑器
- **Luckysheet**: 功能强大的在线表格编辑器
- 支持打开/保存 Excel 文件 (.xlsx, .xls, .csv)
- 支持公式、格式化、图表等功能

### 📝 文档编辑
- **Quill 富文本编辑器**
- 支持打开 Word 文档 (.docx)
- 支持富文本格式：标题、字体、颜色、列表、对齐等
- 导出为 HTML 格式

### 🎨 演示文稿
- **PptxGenJS**
- 创建和编辑 PowerPoint 演示文稿
- 添加/删除幻灯片
- 导出为 .pptx 格式

### 🧠 思维导图
- **D3.js 实现**
- 可视化层次结构思维导图
- 点击节点可编辑内容
- 导出为 JSON 格式

### 📐 图表设计
- **mxGraph / Canvas**
- 创建流程图、架构图等
- 支持矩形、椭圆、菱形、三角形等形状
- 导出为 XML 或 PNG 格式

## 使用方法

1. 从左侧菜单选择文档类型
2. 点击"新建"创建新文档，或"打开"加载现有文件
3. 在编辑区域进行编辑
4. 点击"保存"或"导出"保存文档

## 依赖库

已下载到 `lib/` 目录：
- jQuery 3.7.1
- JSZip
- SheetJS (xlsx)
- x-spreadsheet
- Luckysheet
- Quill
- Mammoth
- docx
- PptxGenJS
- D3.js
- mxGraph

## 注意事项

- 所有处理均在浏览器本地完成，数据不会上传到服务器
- Word 文件导出目前仅支持 HTML 格式，完整 DOCX 导出需要服务器支持
- SVG 图标加载失败不影响功能使用
- 思维导图数据以 JSON 格式保存
