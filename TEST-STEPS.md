# 局域网聊天快速测试步骤

## 重要提示
⚠️ **只启动一个服务器实例！** 所有用户都连接到同一个服务器。

## 测试步骤

### 1. 启动服务器
```bash
# 只启动一个
DataToolbox-Server.exe
```

看到类似输出：
```
============================================================
DataToolbox 服务器已启动
============================================================
本地访问: http://localhost:8080
局域网访问: http://192.168.1.xxx:8080
============================================================
```

### 2. 单机测试（用多个浏览器窗口）

#### 打开第一个用户
1. 打开浏览器窗口A
2. 访问 `http://localhost:8080/apps/lan-chat/`
3. 点击左上角昵称，设置为"用户A"
4. 看到状态变为"在线"（绿点）

#### 打开第二个用户
1. **新开一个浏览器窗口B**（不要用标签页，用独立窗口）
2. 访问 `http://localhost:8080/apps/lan-chat/`（同一个地址！）
3. 点击左上角昵称，设置为"用户B"
4. 看到状态变为"在线"

#### 验证互相发现
- 窗口A左侧列表应该看到"用户B"
- 窗口B左侧列表应该看到"用户A"
- 如果看到了，成功！✅

#### 测试聊天
1. 在窗口A点击"用户B"
2. 发送消息"你好"
3. 切换到窗口B，应该收到消息
4. 在窗口B回复"你好呀"
5. 切换到窗口A，应该收到回复

### 3. 多设备测试

#### 准备工作
- 确保所有设备在同一WiFi网络
- 记下服务器的局域网IP（启动时会显示）

#### 设备1（服务器电脑）
访问: `http://localhost:8080/apps/lan-chat/`

#### 设备2（手机/平板/其他电脑）
访问: `http://192.168.1.xxx:8080/apps/lan-chat/`
（将xxx替换为实际IP）

#### 设备3+
同样使用局域网IP访问

### 4. 功能测试

在两个用户之间测试：
- ✅ 发送文字消息
- ✅ 点击😊发送表情
- ✅ 点击🖼️发送图片
- ✅ 点击📎发送文件
- ✅ 未读消息提醒

## 常见问题

### Q: 我启动了8080和8081两个服务器，为什么看不到对方？
**A**: 这是错误的使用方式！每个服务器都是独立的，无法互相通信。正确方式是只启动一个服务器，所有用户都连接到它。

### Q: 浏览器标签页能当多用户吗？
**A**: 可以，但建议用独立窗口，因为有些浏览器会共享标签页的LocalStorage，可能导致昵称冲突。

### Q: 如何在手机上测试？
**A**: 
1. 手机连接到与电脑相同的WiFi
2. 在浏览器输入电脑的局域网IP
3. 例如: `http://192.168.1.100:8080/apps/lan-chat/`

### Q: 看不到其他用户怎么办？
**A**: 检查：
1. 是否访问的是同一个服务器地址？
2. 状态是否显示"在线"（绿点）？
3. 打开浏览器控制台，看是否有错误？
4. 刷新页面重试

### Q: 消息发不出去
**A**: 检查：
1. 网络连接是否正常？
2. WebSocket是否连接成功（状态应该是"在线"）？
3. 对方是否还在线？

## 架构说明

```
┌─────────────────────────────────────┐
│    DataToolbox Server (8080)        │
│    ┌─────────────────────────┐     │
│    │         Hub             │     │
│    │  (管理所有客户端连接)    │     │
│    └─────────────────────────┘     │
│         ↑    ↑    ↑    ↑           │
└─────────┼────┼────┼────┼───────────┘
          │    │    │    │
    ┌─────┘    │    │    └─────┐
    │          │    │          │
┌───┴──┐  ┌───┴──┐ ┌──┴───┐ ┌─┴────┐
│ 窗口A │  │ 窗口B │ │ 手机  │ │ 平板 │
└──────┘  └──────┘ └──────┘ └──────┘
```

**正确**: 所有设备连接到同一个服务器
**错误**: 启动多个服务器，设备分别连接
