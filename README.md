# DataToolbox - 数据工具箱

一个功能丰富的离线数据处理和学习工具集合，现已支持局域网联机功能。

## ✨ 特色功能

### 🌐 联机分区（新增）
- **局域网聊天**: 类微信的即时通讯应用
  - 自动设备发现
  - 实时消息传输
  - 支持文字、表情、图片、文件传输
  - 响应式设计，支持移动端

### ⚡ 效率工具
- **运维工具箱**: SSH终端、SFTP、网络诊断（Ping/DNS/HTTP/端口检测）
- **字段匹配**: 智能字段映射
- **AI结构化**: 非结构化文本转JSON
- **格式转换**: 图片、文档格式互转（Excel/CSV/JSON/PDF/Word）
- **文档编辑器**: Excel/Word/PPT/思维导图/Visio在线编辑
- **待办清单+番茄钟**: GTD任务管理与专注计时
- **刷题练习**: Excel导入题库，支持多种题型
- **低代码开发**: 可视化拖拽构建数据应用（开发中）

### 🎓 教学分区
- **数据库学习**: SQL语法、MyBatis映射闯关式学习
- **Web开发学习**: HTTP请求、API接口、HTML工具开发
- **文件处理学习**: Excel/Word处理、JS文件操作
- **运维学习**: 命令行操作、Docker、数据库运维
- **AI学习**: 大模型使用、Prompt工程、攻防挑战
- **算法学习**: 数据结构、算法、机器学习
- **项目管理**: 甲方素养、需求表达、防坑指南

## 🚀 快速开始

### 方式一：直接使用（纯前端功能）
1. 下载项目到本地
2. 用浏览器打开 `index.html`
3. 选择工具开始使用

> ⚠️ 注意：联机功能需要服务器版本

### 方式二：服务器版本（支持联机功能）

#### 1. 安装依赖
```bash
# 安装 Go 环境（如未安装）
# 访问 https://golang.org/dl/

# 下载依赖
go mod download
```

#### 2. 构建服务器
```bash
# Windows
go build -o DataToolbox-Server.exe server.go

# Linux/Mac
go build -o DataToolbox-Server server.go
```

#### 3. 启动服务器
```bash
# Windows
DataToolbox-Server.exe

# Linux/Mac
./DataToolbox-Server
```

#### 4. 访问应用
- 本地访问: `http://localhost:8080`
- 局域网访问: `http://<局域网IP>:8080`

详细指南请查看 [QUICKSTART.md](QUICKSTART.md)

## 📱 局域网聊天使用方法

1. 启动服务器版本
2. 访问聊天应用: `http://localhost:8080/apps/lan-chat/`
3. 设置你的昵称
4. 等待局域网内其他设备连接
5. 选择联系人，开始聊天！

支持功能：
- ✅ 文字消息
- ✅ 表情符号（80+ 个）
- ✅ 图片发送（支持粘贴、拖拽）
- ✅ 文件传输（支持拖拽、批量发送）
- ✅ 已读未读状态
- ✅ 在线状态显示
- ✅ **拖拽发送**：拖拽文件到窗口直接发送
- ✅ **粘贴发送**：Ctrl+V 粘贴图片直接发送
- ✅ **批量发送**：支持多文件拖拽批量发送

详细说明请查看 [apps/lan-chat/README.md](apps/lan-chat/README.md)

## 🔧 自定义配置

### 修改端口
```bash
# 命令行参数
DataToolbox-Server.exe -port 9000

# 或编辑 server.config.json
{
  "port": 8080,
  "host": "0.0.0.0"
}
```

### 修改监听地址
- `0.0.0.0`: 允许局域网访问（默认）
- `127.0.0.1`: 仅本地访问

## 🐛 常见问题

### Q: 为什么局域网聊天提示需要服务器版本？
A: 局域网聊天使用WebSocket协议，必须通过服务器运行。请按照[快速启动指南](QUICKSTART.md)构建并启动服务器。

### Q: 其他设备无法连接到聊天？
A: 请检查：
1. 所有设备在同一局域网
2. 防火墙允许端口访问
3. 使用正确的局域网IP地址

### Q: 消息会保存吗？
A: 当前版本消息仅保存在浏览器内存中，刷新或关闭页面后会丢失。未来将支持消息持久化。

### Q: 支持多人群聊吗？
A: 当前版本仅支持一对一聊天，未来将添加群聊功能。

更多问题请查看 [apps/lan-chat/README.md](apps/lan-chat/README.md)
